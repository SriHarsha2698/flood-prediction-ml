# 🌊 Flood Prediction using Ensemble Machine Learning

## 📌 Overview

This project predicts flood probability using ensemble machine learning models such as Random Forest, XGBoost, and LightGBM based on environmental and societal factors.

## 🚀 Key Results

* XGBoost: R² = 95.51%
* LightGBM: R² = 94.98%
* Random Forest: R² = 78.51%

## 📊 Dataset

* Source: Kaggle
* Records: 50,000
* Features: 21 (expanded to 26 after feature engineering)

## 🧠 Feature Engineering

Key engineered features:

* Environmental Pressure
* Infrastructure Vulnerability
* Preparedness Deficiency
* Population Environmental Stress

## ⚙️ Models Used

* Random Forest
* XGBoost
* LightGBM

## 📈 Evaluation Metrics

* RMSE
* R² Score
* Bias-Variance Tradeoff
* Training & Inference Time

## ▶️ How to Run

1. Open the notebook in Google Colab
2. Upload dataset (`flood.csv`)
3. Run all cells

## 📌 Key Insights

* XGBoost achieved the best performance
* Feature engineering significantly improved accuracy
* Environmental & societal factors strongly influence flood risk

## ⚠️ Limitations

* Dataset constraints
